Ejercicio N1 del PreapCourse

Siguiendo todas las instrucciones y los lineamientos aprendidos en el PreapCourse, ya estoy lista para crear una pagina simple desde 0

Los pasos a seguir son:

  • Abrir una nueva carpeta donde deseo guardar el trabajo
  • Crear un archivo en la carpeta y darle el nombre correspondiente
  • Escribir la estructura basica

    • Una vez finalizado el procedimiento, hay que subir el trabajo a GITHUB

    Quizas parezca dificil pero te dejare los pasos a seguir

    1. Abre la terminal y ejecuta el comando
    2. Crea un repositorio
    3. Crea un nuevo repositorio
    4. Sigue las instrucciones proporcionadas por GITHUB
    5. Vincula al reposorio
    6. Actualiza y asegurate que los cambios se hayan guardado correctamente
